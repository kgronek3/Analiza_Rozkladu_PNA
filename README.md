# Analiza rozkładu wypadków samochodowych w Polsce
Niniejsze repozytorium zawiera projekt na zajęcia z Programowania Narzędzi Analitycznych I na WNE UW. Projekt przedstawia analizę rozkładu wypadków samochodowych w Polsce dla danych dziennych od 2009 do 2022 roku. Oszacowanie parametrów rozkładu będzie dokonane metodą największej wiarygodności (MNW) oraz uogólnionej metody momentów (UMM).
